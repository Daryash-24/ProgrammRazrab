# Чек-лист с наборами данных для позитивного и негативного тестирования
# Позитивные тесты

1. (3, 3, 3) => equilateral (Равносторонний треугольник)
2. (5, 5, 7) => isosceles (Равнобедренный треугольник)
3. (3, 4, 5) => nonequilateral (Разносторонний треугольник)
4. (2.5, 3.5, 4.5) => nonequilateral (Разносторонний треугольник)

# Негативные тесты

1. (0, 0, 0) => IncorrectTriangleSides: Все стороны должны быть положительными
2. (-1, 0, 2) => IncorrectTriangleSides: Все стороны должны быть положительными
3. (0, 1, 1) => IncorrectTriangleSides: Все стороны должны быть положительными
4. (1, 1, 3) => IncorrectTriangleSides: Не выполняется неравенство треугольника
5. (2, 2, 4) => IncorrectTriangleSides: Не выполняется неравенство треугольника
